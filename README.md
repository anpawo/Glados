# GLaDOS
GLADOS is a project where we will implement a programming language in Haskell. The project is divided into different parts, each of which will progressively add features to our language. Here's an overview of the main requirements and steps.

By the end of the project, GLADOS will be a functional, minimalist programming language with the ability to handle basic operations, conditional logic, and user-defined functions. Through the first part, you will create the core features of the language, and in the second part, you will enhance it to make it more powerful and secure.

## Getting Started

### Installation

Install Stack and GHC.
```sh
sudo apt update
sudo apt install haskell-stack
sudo apt install ghc

stack --version
ghc --version
```

Clone this repository.
```sh
git clone git@github.com:anpawo/Glados.git
```

### Quickstart
Build the project:
```sh
make
```

### Usage
```sh
./glados ...
```
